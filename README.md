Desarrollar un Sistema Web para optimizar la Gestión de citas en pacientes de una Clínica

Entregable 1 del Proyecto final del Curso de MARCOS DE DESARROLLO WEB

Nombre del proyecto:
Desarrollar un Sistema Web para optimizar la Gestión de citas en pacientes de una Clínica en lima, 2026
Objetivo
Desarrollar un sistema web que permita:
•	Registrar usuarios (clientes/pacientes) 
•	Gestionar citas 
•	Administrar servicios 
•	Controlar accesos con seguridad 
Integra front-end, backend, BD y seguridad
Entregable 1:
Nombre: Prototipo de interfaz de citas
🔹 Contenido:
•	Página principal 
•	Formulario de registro/login 
•	Agenda visual de citas 
•	Uso de Bootstrap: 
o	Grid responsive 
o	Cards (servicios) 
o	Modals (detalle de cita) 
🔹 Funcionalidad:
•	Simulación de registro de citas (JS) 
•	Validaciones básicas 
🔹 Resultado:
Sistema visual navegable (sin backend)
